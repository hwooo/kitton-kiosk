## Kitton-Kiosk
가상의 키오스크 시스템 `키튼키오스크` 백엔드 소스 코드입니다.

### 기술 스택
- TypeScript
- NestJS
- TypeORM
- GitHub
- AWS
  - Elastic Beanstalk
  - EC2
  - RDS MySQL 8.0
  - CodePipeline
  - Route 53
